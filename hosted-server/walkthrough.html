Next Steps
#1: Download and install python for your platform
The software that runs on your computer is written in a language called python and must be run by another program called an interpreter. There is a version of the python interpreter for most operating systems.
Linux Users: You probably already have python installed. It's pretty standard. Type python and if it starts up press Ctrl+D to quit and skip this section.
Windows Users: Go to http://www.python.org/download and select the Windows installer that suits your platform (32 or 64 bit.) Run it, accept all the defaults.
Mac Users: You already have it!
#2: Download and unzip the Server Software
#3: Start the server
Linux Users: 
	    cd (installation)/server
	    python mymedia.py
Windows Users: Double click on mymedia.bat in the installation directory.
Mac Users: Ctrl+Click on mymedia.py in the (installation)/server directory and select Open With -> Python Launcher.
#4: Configure Your Server
Click on the link above to reach your server's configuration page. This is where you tell MyMedia where to find your music, videos, and photos. Verify that your server's IP address was autodetected correctly as well. Some computers with more than one network card may not have their IP address autodetected correctly.
#5: Enjoy!

My Media by el.wubo

A client and server combo for serving mp3s and other local media over your local network to your Roku set-top box.

Dependencies

Python

Install

  1. Download and install python for your platform
  2. Download MyMedia
  3. Put your Roku into developer mode
  4. Run springboard.vbs to install the channel on your Roku and start the server
